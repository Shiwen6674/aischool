# Example: Promote Placeholder Page

## When To Use

你要把目前空白的 `science_cap.html`、`student_english.html` 之類頁面升級為真正可用的功能頁。

## Suggested Prompt

```text
請依 AGENTS.md 與 docs/workflows.md 的「Promote A Placeholder Page」流程，
把 aischool 的 `science_cap.html` 升級成最小可用頁面。
請先確認它從哪個頁面被導入，再補齊內容、登入相容性、必要導覽與文件。
```

## Good Output Looks Like

- 先確認入口來源
- 學生內容頁優先沿用 `assets/css/student-hub.css` 與 `assets/js/student-hub.js`
- 不會只做靜態畫面而忘了導頁與 session
- 文件裡的 placeholder 狀態會同步更新
